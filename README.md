# HR-Analytics-PowerBI-Dashboard
Interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, satisfaction, salary distribution, and workforce trends.
## 🛠️ Tools & Technologies

- Google Colab (Python)
- Pandas
- NumPy
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- CSV Dataset
- Git & GitHub

---

## 🔄 Project Workflow

1. Imported the raw HR dataset into **Google Colab**.
2. Performed data cleaning and preprocessing using **Python (Pandas)**.
3. Handled missing values, corrected data types, and prepared the dataset for analysis.
4. Exported the cleaned dataset as a CSV file.
5. Imported the cleaned dataset into **Microsoft Power BI**.
6. Created DAX measures for key HR metrics.
7. Designed an interactive dashboard with KPI cards, charts, and slicers.
8. Published the project on GitHub for portfolio purposes.
